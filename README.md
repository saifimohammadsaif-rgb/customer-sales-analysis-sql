# Customer Sales Analysis — SQL Project

## About This Project
This project analyzes customer data using SQL to answer 
real business questions about customer distribution 
across countries and cities.

## Tools Used
- SQL Server
- W3Schools Database

## Business Questions Answered
1. How many customers do we have in total?
2. Which country has the most customers?
3. Show all German customers sorted A to Z
4. How many customers are in each city?
5. Which countries have more than 3 customers?
6. Top 5 countries with most customers
7. Countries starting with letter G
8. Countries with between 3 and 7 customers

## SQL Skills Demonstrated
- SELECT and FROM
- WHERE filtering
- GROUP BY and COUNT
- HAVING clause
- LIKE for pattern matching
- BETWEEN for ranges
- ORDER BY ASC and DESC
- TOP 5 for limiting results
